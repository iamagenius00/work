# Antenna Event — 让你的活动参与者找到对的人

## Antenna 是什么

Antenna 是一个轻量的现场社交发现工具。参与者加入活动后，可以看到"现场还有谁"，找到值得聊的人，双向匹配后交换联系方式。

没有 App，没有注册，没有社交图谱。一个活动码走天下，活动结束自动清除。

## 活动主办方可以用 Antenna 做什么

### 1. 创建专属活动

主办方创建活动，获得一个唯一的活动码（如 `ad49594f`）。把这个码分享给参与者就行——印在海报上、发在群里、贴在签到台。

每个活动有独立的活动页：`antenna.fyi/e/活动码`，支持自定义活动名称、描述和封面图。

### 2. 参与者一码加入

参与者拿到活动码后，一个指令加入活动。不需要下载 App，不需要注册账号。

加入后，AI agent 会根据你的背景自动生成一张名片：一个 emoji、一个名字、三行自我介绍。你确认一下就行，不用自己写。

### 3. 现场扫描 — "这里还有谁"

参与者在活动现场可以扫描，看到同一活动里的其他人。每个人显示名片信息，主办方有专属标识。

这不是一个参与者名单——是一个"此刻在现场的人"的实时快照。

### 4. 智能匹配 — "谁值得聊"

Antenna 会根据双方名片内容做语义匹配，告诉你"这个人跟你可能聊得来，因为______"。

不是随机推荐，是基于你写的三行介绍做的理解。写得越具体，匹配越准。

### 5. 双向确认，交换联系方式

看到感兴趣的人，点"想认识"。对方也点了，双向匹配成功，自动交换联系方式（微信、Twitter、邮箱——参与者自己选择分享什么）。

单向的不会通知对方，没有社交压力。

### 6. 签到追踪

区分"已报名"和"已到场"。主办方可以实时看到：多少人加入了活动，多少人真的到了现场。到场的参与者旁边有 ✅ 标记。

### 7. 活动结束，数据清除

主办方可以随时结束活动。结束后不能再加入，数据自动清除。没有历史记录，没有社交债务。

## 一场活动的完整流程

```
活动前
  主办方 → 创建活动，拿到活动码
  主办方 → 把码分享给参与者（海报/群聊/邀请函）

活动中
  参与者 → 用活动码加入，填名片
  参与者 → 到现场后签到
  参与者 → 扫描现场的人，看谁值得聊
  参与者 → 对感兴趣的人点"想认识"
  双方都点了 → 交换联系方式 🎉

活动后
  主办方 → 结束活动
  数据自动清除，干干净净
```

## 适合什么活动

- Hackathon / Hackerhouse — 帮开发者找到互补的队友
- 行业meetup — 100+人的场子，不可能每个人都聊到，Antenna 帮你找到最值得聊的那几个
- 闭门沙龙 — 小范围高质量社交，双向匹配过滤掉无效社交
- 发布会 / Demo Day — 让观众之间也能互相发现

## 为什么不用微信群

| | 微信群 | Antenna |
|---|---|---|
| 找人 | 滚聊天记录，靠缘分 | 智能匹配，告诉你为什么值得聊 |
| 隐私 | 加了群就暴露微信号 | 双向匹配才交换，你选择分享什么 |
| 活动后 | 变成僵尸群或广告群 | 自动清除，没有后续负担 |
| 破冰 | 自己想开场白 | "Antenna 说我们可能聊得来"就是最好的开场白 |

## 技术接入

Antenna 支持 CLI、MCP Server 和 API 三种接入方式。如果你的活动有 AI agent 做助手，agent 可以直接帮参与者扫描、匹配、签到——全程自动化。
